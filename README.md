## Setup

0. Make sure you have the latest `node` and `npm` installed:
  ```shell
  node --version
  npm --version
  ```

1. Clone the repo:
  ```shell
  git clone https://github.com/qiuqili/falls-hotel.git
  cd falls-hotel
  ```

2. Install `npm` dependencies:
  ```shell
  npm install
  ```

3. Start `BrowserSync` with `gulp`:
  ```shell
  npm start
  ```

4. Start developing! :avocado:
